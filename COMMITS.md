# Convención de commits

```
(alcance en inglés): descripción en español
```

## Ejemplos

```
(feat): Agregar campo type a reportes de problema
(fix): Corregir validación de email en registro
(refactor): Mover lógica de topbar a TopbarComposer
(docs): Actualizar README con instrucciones de instalación
(ui): Ajustar estilos del modal de reportes
(test): Agregar tests para ProjectService
(chore): Actualizar dependencias de composer
```
